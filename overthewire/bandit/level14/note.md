# 출제의도

어떤 포트가 열려있는 지 확인하고 해당 포트에 데이터를 보내는 문제를 낸 것으로 보임.

# 배운 점

nmap vs nc
nmap: 호스트의 오픈 포트 탐지
nc: 포트의 상태 확인 가능, 데이터 전송 가능

`echo <CURRENT PASSWORD> | nc localhost 30000`와 같이 nc접속 후 데이터 전송이 아닌 파이프를 통한 바로 전송되는 부분도 확인함.
