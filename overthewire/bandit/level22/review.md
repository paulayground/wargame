# 배운 점

`whoami`가 환경변수에 저장되서 출력되는 값인줄 알았는데, 현재 로그인한 유저를 출력하는 프로그램이였다.
`md5sum`할 때 마지막에 `-` 이 왜 나오는지 몰랐는데,
파일 이름으로 받을 때와 달리 표준입력을 통해 받았을 때 `-`것이 나오는 것을 알게됐다.

```bash
md5sum test.md
8ca319486bfbbc3663ea0fbe81326349  test.md

echo "I am user bandit23" | md5sum
8ca319486bfbbc3663ea0fbe81326349  -
```

# 풀이 과정

`whoami`가 환경변수에 저장되는 값인 줄 착각하여 환경변수를 바꿔봤지만 `whoami`값이 바뀌지 않는다는 것을 알고 이상해 찾아본 결과 아니였다.
환경변수를 변경해서 해결하는 방법이 아니라고 깨닫고 다른 md5sum값은 같은 내용이면 같으니까 직접 `bandit23`인 이름으로 변경해서 실행하면 될 것이라고 생각했다.

# 익스플로잇 코드 정리

```
# "I am user bandit23"값을 md5로 인코딩
# 인코딩한 값에 공백(' ')을 구분자(-d) 기준으로 자르고 첫번째(1) 필드(-f)만 출력
echo "I am user bandit23" | md5sum | cut -d ' ' -f 1
```

# 한 줄 평

잘 생각해보면 whoami가 환경변수가 아니라는 것을 알았을텐데 무의식적으로 맞다고 착각한 것같다. 하나하나 파고드니까 조금씩 모르는 부분들이 나오는 것 같다.
